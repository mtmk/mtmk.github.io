Asp.Net CGI handler

I created Asp.Net CGI Handler to enable some of the old Perl scripts to run under Asp.Net.

CGI Handler is a Asp.Net HttpHandler implementation which can run programs as CGIs.

Code is meant to be an example; there are no binaries or releases. If you want to use it, you will need to copy the CgiHttpHandler.cs into your project and modify a few things to get it running.

Enjoy!
